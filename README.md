🧱 B-Tiles - Aesthetic Tiles Gallery Portal
B-Tiles is a modern, high-performance web application designed to showcase a premium collection of aesthetic tiles. Built with Next.js (App Router), it provides a seamless experience for users to browse, search, and manage their favorite tile designs.


✨ Key Features
Responsive Design: Fully optimized for Mobile, Tablet, and Desktop views using Tailwind CSS and DaisyUI.

Dynamic Tile Gallery: Browse an extensive list of tiles with real-time search functionality by tile title.

Authentication System: Secure user authentication (Email/Password & Google Social Login) powered by BetterAuth and MongoDB.

Private Routes: Secure access to Tile Details and User Profile pages, restricted to logged-in users only.

User Profile Management: Logged-in users can view their profile data and update their Name and Image URL.

Interactive UI: Includes a dynamic banner, marquee text for new arrivals, and smooth loading states.

Custom Error Handling: Implementation of a "Not Found" (404) page and proper error messages for failed login/registration.
